# Healthcare Power BI Dashboard

# Project Overview
This project analyzes healthcare data using Power BI to track patient admissions, diagnosis trends, billing, and hospital resource utilization.

# Tools Used
- Power BI
- Excel / CSV Dataset

# Key Features
- Patient diagnosis analysis
- Insurance vs billing comparison
- Bed occupancy tracking (ICU, Private, General)
- Doctor feedback analysis

# Objective
To provide actionable insights for hospital management using interactive dashboards.
